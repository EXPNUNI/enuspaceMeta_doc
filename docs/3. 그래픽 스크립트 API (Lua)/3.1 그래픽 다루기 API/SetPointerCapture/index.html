Lua API

SetPointerCapture() 

- 함수를 실행한 객체를 이벤트 캡처 대상으로 지정합니다. 후속 이벤트는 캡처가 해제될 때 까지(mouseup 이벤트) 캡처 요소를 대상으로 합니다.

Parameter (인자) 

- none

Return (반환값) 

- none

Remark (주의사항) 

- mouseup 이벤트 이후 캡처가 해제됩니다.

사용예시 

function _onmousedown()
  	SetPointerCapture()
    bIsMouseDown = true;
end

function _onmousemove()
	if(bIsMouseDown) then
        translate_x = GetLocalCursorPosX()
    end
end

function _onmouseup()
	bIsMouseDown = false
end
